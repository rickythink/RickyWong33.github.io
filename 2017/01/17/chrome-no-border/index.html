Chrome设置无边框的方法

发表于 2017-01-17 | | 阅读次数

教你如何在Windows7上设置Chrome无边框的方法

我们时常能在国产浏览器上看到非常出彩的设计，比如猎豹：

比如360极速浏览器：

对比Windows上的Chrome浏览器：

可以发现，国产的浏览器在界面简洁性上似乎更胜一筹。究其原因，我认为是边框的错！

Chrome浏览器由于继承了Windows默认的边框边距设定，所以看起来的界面视野没有国产浏览器那么开拓。下面简单介绍一下如何设置无边框的Chrome浏览器。

打开个性化设置

在桌面右键 -> 个性化

选择窗口颜色选项

选择高级外观设置

选择外框填充项目

设置边框为0

应用 - 确定后再看一下，是不是Chrome浏览器变得更加清爽了呢？
